# Password Manager with Zero-Knowledge Architecture

## Project Overview

This project is a secure Password Manager developed in Python. The system follows a Zero-Knowledge Architecture where user credentials are encrypted locally and plaintext passwords are never stored.

## Features

* Master Password Authentication
* AES-256-GCM Encryption
* Argon2 Key Derivation
* Password Strength Checker
* Breach Detection using HaveIBeenPwned API
* Auto-Fill Functionality
* Secure Local Storage

## Technologies Used

* Python
* Tkinter
* Argon2
* AES-256-GCM
* JSON
* Pyperclip
* HaveIBeenPwned API
